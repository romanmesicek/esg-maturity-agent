# Output

Verzeichnis für generierte Analyseberichte.

## Struktur

Für jede Analyse wird ein Unterordner erstellt:

```
output/
└── [firmenname]_[datum]/
    ├── analyse_roh.md
    └── analyse_bericht.html
```

## Dateien pro Analyse

| Datei | Beschreibung |
|-------|--------------|
| `analyse_roh.md` | Rohdaten, Volltext, alle Quellen |
| `analyse_bericht.html` | Formatierter Bericht mit Radar-Diagramm |

## Hinweis

**Inhalte dieses Verzeichnisses werden NICHT ins Repository hochgeladen.**

Analyseberichte enthalten unternehmensspezifische Daten und werden nur lokal gespeichert.
