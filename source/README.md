# Source

Referenzmaterial für das Ainsbury-Grayson Framework.

## Im Repository enthaltene Dateien

### ainsbury_grayson_referenz.md

Umfassende deutschsprachige Referenzdokumentation mit:

- Theoretischer Hintergrund und akademische Einordnung
- Die 5 Reifegradstufen im Detail
- Die 11 Embedding-Kategorien
- **APPENDIX 1:** Detaillierte Stufenmatrix (A1.1-A1.11)
- **APPENDIX 2:** Funktionale Bereiche (A2.1-A2.6)
- **APPENDIX 3:** Dunphy's Vergleichsmodell

### ainsbury_grayson_top-level_categories.md

Kompakte Übersicht der 11 Bewertungskategorien.
