# Input-Ordner für Nachhaltigkeitsanalysen

Dieser Ordner enthält Kundenmaterialien für die Nachhaltigkeitsanalyse.

## Struktur

Für jedes zu analysierende Unternehmen einen Unterordner anlegen:

```
input/
├── firmenname_a/
│   ├── nachhaltigkeitsbericht_2024.pdf
│   ├── geschaeftsbericht_2024.pdf
│   └── pressemitteilungen/
├── firmenname_b/
│   └── ...
```

## Empfohlene Materialien

| Material | Priorität | Beschreibung |
|----------|-----------|--------------|
| Nachhaltigkeitsbericht | Hoch | CSR/ESG-Bericht des Unternehmens |
| Geschäftsbericht | Hoch | Jahresbericht mit Nachhaltigkeitskapitel |
| GRI-Index | Mittel | Falls separat veröffentlicht |
| CDP-Fragebogen | Mittel | Climate Disclosure Antworten |
| Pressemitteilungen | Niedrig | Relevante Nachhaltigkeits-News |
| Website-Screenshots | Optional | Nachhaltigkeitsbereich der Website |

## Namenskonvention

- Ordnernamen: Kleinbuchstaben, Unterstriche statt Leerzeichen
- Beispiel: `deutsche_bahn`, `siemens_energy`, `basf`

## Hinweis

Der Agent führt zusätzlich Web-Recherche durch. Bereitgestellte Materialien werden priorisiert und ergänzen die Online-Quellen.

**Inhalte dieses Verzeichnisses werden NICHT ins Repository hochgeladen.**
